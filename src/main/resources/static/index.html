Sail-Jinx

A companion application to SailSys that provides an alternate race processing UI and the Jinx handicap solution for pursuit races. After each race, a boat's handicap is adjusted to achieve the effect on their next starting time as follows:

  • Top finishers receive fixed time penalties by place. For example: 6 minutes for first, 4 for second, 2 for third, 1 for fourth. This is added to their next start time.
  • All starters receive a variable time bonus equal to the total time penalties given (e.g. 6+4+2+1), distributed so that slower boats get a larger share, especially in longer races.
  • Fleet adjusted for the earliest starter so that the slowest boat starts exactly on the earliest allowed start time.
